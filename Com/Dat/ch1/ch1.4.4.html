1.4.4 C++의 데이터 선언

열거 타입: 일련의 정수 상수를 선언하는 또 다른 기법이다. 또한 열거 타입에 이름을 지정하여 새로운 데이터 타입을 만들기 위해 쓰인다.

enum semester{SUMMER, FALL, SPRING};

SUMMER, FALL, SPRING은 semester 타입의 정수 상수이다. 이들의 값은 기정 값으로서 각각 0, 1, 2,이다.

포인터: 객체의 메모리 주소를 저장한다.

int i = 25;
int *np;
np = &i;

참조 타입: 객체에 대한 또 다른 이름을 제공한다.

int i = 5;
int& j = i;
i = 7;
printf(“i=%d, j=%d”, i, j);
i의 값이 바뀌면 j의 값도 따라서 바뀐다. 따라서 printf 문이 실행될 때 i와 j의 최종 값은 7이 된다.
