1.4.6 C++의 함수 이름 다중화

함수 다중화(function overloading)는 함수의 시그니처(인자 리스트)가 다르기만 하면 같은 이름을 가진 함수가 둘 이상 존재할 수 있다는 뜻이다.

즉 아래와 같은 선언이 허용된다.

int Max(int, int);
int Max(int, int, int);
int Max(int*, int);
int Max(float, int);
int Max(int, float);
