1.4.7 인라인 함수

inline int Sum(int a, int b)
{
  return a + b;
}

이 inline 키워드는 Sum에 대한 모든 호출이 Sum의 몸체로 대체되어야 한다는 것을 컴파일러에게 알린다. 이는 함수 호출 및 인자 복사 오버헤드를 줄인다. 또 #define의 필요성을 없애 디버그를 용이하게 만든다.
